# mms-deps-watcher
GoodData dependencies watcher for AD, KD (not tested yet) projects.

## Supported libraries

* gdc-app-components
* gooddata-react-components
* gooddata-indigo-visualizations
* gooddata-data-layer
* gooddata-js
* gdc-goodstrap

## Install & Run

Works automatically with yarn command.

### Run watchers
```
gulp
```
